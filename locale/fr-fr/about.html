Cette carte interactive présente les stationnements pour cyclistes de l'agglomération tourangelle.

Des données

Les données sur les parkings vélo sont issues d'OpenStreetMap, une base de données géographique libre et coopérative. OpenStreetMap recense une multitude d'équipements, de voies de circulation, de commerces à travers le monde, sans oublier les stationnement pour cyclistes.

Les données sont mises à jour toutes les heures depuis OpenStreetMap.

Vous pouvez aider à améliorer cette cartographie, en ajoutant des notes sur OpenStreetMap (bouton "Suggérer une amélioration de la carte"), ou bien en modifiant vous-même les données d'OpenStreetMap.
